# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

## Файл rectangle.py
- Функция `area(a, b)` принимает два целых числа a и b - длины сторон - и возвращает их произведение - площадь прямоугольника
```py
area(4, 5) # -> 20
```
- Функция `perimeter(a, b)` принимает два целых числа a и b - длины сторон - и возвращает их удвоенную сумму - периметр прямоугольника
```py
perimeter(4, 5) # -> 18
```

## Файл circle.py
- Функция `area(r)` принимает целое число r - радиус круга - и возвращает π * r * r - площадь круга
```py
area(5) # -> 78.53982
```
- Функция `perimeter(r)` принимает целое число r - радиус круга (окружности) - и возвращает 2 * π * r - периметр круга (длину окружности)
```py
perimeter(5) # -> 31.41593
```

## Файл square.py
- Функция `area(a)` принимает целое число a - длину стороны - и возвращает a * a - площадь квадрата
```py
area(6) # -> 36
```
- Функция `perimeter(a)` принимает целое число a - длину стороны - и возвращает 4 * a - периметр квадрата
```py
perimeter(6) # -> 24
```
